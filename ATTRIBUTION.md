# Attributions and Resources

## Datasets
* **NYC Smells Dataset:** Provided via Voxel51, containing 7,000+ paired environmental images and electronic nose (Cyranose 320) sensor readings.

## Academic & Technical Foundations
* **Latent Diffusion Models (LDM):** Generative architecture adapted from Rombach et al. (2022).
* **CLIP (Contrastive Language-Image Pre-training):** Theoretical basis for the COIP alignment stage (Radford et al., 2021).

## Libraries & Tools
* **PyTorch:** Primary framework for neural network architecture and training.
* **Hugging Face Diffusers:** Core library for DDPM noise schedulers and diffusion sampling.
* **Transformers (Hugging Face):** Implementation of the Vision Transformer (ViT) backbone.
* **Scikit-Learn:** Used for robust object-level data stratification and metric calculation.
* **Openpyxl/Pandas:** Utilized for managing metadata and experimental results.

## AI
* **Gemini:** Generation of documents and citations. Contributed during debugging.
